# Soft ID - Startup Landing Page

## 🛠️ Tech Stack

- **HTML5**: Struktur semantik yang ramah SEO.
- **Tailwind CSS v3+**: Framework utilitas CSS untuk kustomisasi cepat.
- **Inter / System Sans Serif**: Tipografi bersih khas platform SaaS.

## ⚡ Panduan Pengembangan Lokal
Untuk menjalankan dan melakukan kompilasi otomatis Tailwind CSS pada proyek ini, ikuti langkah-langkah berikut:

Inisialisasi dan pasang Tailwind CSS melalui terminal:
```
npm install tailwindcss @tailwindcss/cli
```
